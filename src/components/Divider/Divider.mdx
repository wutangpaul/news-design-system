# Divider

A horizontal or vertical rule for separating content, with an optional centered label
(e.g. "OR" between two actions, or a section label like "MORE FROM THIS SECTION").

## Do's and don'ts

- Do use a labelled divider for "OR" splits between alternative actions (e.g. sign-in
  options) instead of plain text next to a rule.
- Don't use `Divider` purely for visual spacing — use layout/gap utilities for that, and
  reserve `Divider` for content that genuinely separates two sections.

## Accessibility

- Unlabelled dividers are purely decorative and `aria-hidden="true"`, since the visual
  rule conveys no information assistive tech needs to announce.
- Labelled dividers use `role="separator"` with the label as the accessible name (via
  `aria-label` for string labels), so screen reader users hear the section boundary.
