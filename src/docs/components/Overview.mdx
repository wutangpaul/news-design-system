# Components

Phase 2 of the design system — 31 reusable, accessible React components, complete and verified
(typecheck, lint, tests, Storybook build all green).

**Typography & actions** — Button, Link, Text, Heading, Icon, Divider, Badge, Tag
**Media & display** — Image, Avatar, Card, Skeleton, Spinner, Progress
**Forms** — Input, Textarea, Select, Checkbox, Radio, Toggle, Switch
**Navigation & disclosure** — Tabs, Accordion, Breadcrumbs, Pagination, Dropdown
**Overlay & feedback** — Modal, Drawer, Tooltip, Alert, Toast

Each component's own docs page (in the sidebar below) covers its props, variants, usage
examples, accessibility notes, and do's/don'ts. See `CONTRIBUTING.md` at the project root for
the conventions every component follows (file structure, styling rules, the WCAG 2.2 AA bar).

Built on the [Design Tokens](?path=/docs/design-tokens-overview--docs) and
[Foundations](?path=/docs/foundations-overview--docs) layers above. The next layer,
[Patterns](?path=/docs/patterns-overview--docs), composes these into editorial experiences and
has not been built yet.
