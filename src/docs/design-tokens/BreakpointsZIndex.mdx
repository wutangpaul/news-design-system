# Breakpoints & Z-Index

## Breakpoints

Matches Tailwind's `screens` config directly — `min-width` breakpoints used with the standard
`sm:`/`md:`/`lg:` etc. utility prefixes.

## Z-Index

A documented stacking scale — overlay components reference these **names** via Tailwind's
`z-*` utilities (e.g. `z-modal`), never raw numbers, so stacking order stays predictable as new
overlay components get added.
