# Shadows & Motion

## Shadows

A 5-step elevation scale (`sm` → `xl`), used to separate raised surfaces (Card, Dropdown,
Modal, Tooltip) from the canvas. Subtle by design — this is an editorial product, not a
material-design app, so shadows read as a soft lift rather than a dramatic drop.

All interactive components must respect `prefers-reduced-motion` (see
[Accessibility Standards](?path=/docs/foundations-accessibility-standards--docs)) — durations
and easings above describe the *default* motion, not a requirement that every component
animate.
