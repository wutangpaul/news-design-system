# Foundations

The standards every component and pattern is built against. Where [Design Tokens](?path=/docs/design-tokens-overview--docs)
define raw values, Foundations define the *rules* for using them — the layer that keeps a
500-person newsroom's worth of contributors building consistent, accessible UI without having
to re-derive these decisions per feature.
