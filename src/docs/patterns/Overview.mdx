# Patterns

Phase 3 of the design system — 34 composed editorial experiences built from the
[Components](?path=/docs/components-overview--docs) library, complete and verified (typecheck,
lint, tests, Storybook build all green).

**Editorial** — Article Header, Hero Story, Article Body, Pull Quote, Block Quote, Inline
Image, Image Gallery, Caption, Byline, Author Card, Reading Progress, Reading Time, Related
Articles, Newsletter Signup, Social Sharing, Comments, Live Blog Entry, Breaking News Banner,
Topic Tags, Category Navigation.

**Navigation** — Global Header, Primary Navigation, Secondary Navigation, Mega Menu, Footer,
Search Experience.

**Content Discovery** — Story Card, Featured Story Grid, Most Read, Trending, Recommended
Articles, Topic Collections, Video Cards, Podcast Cards.

Each pattern's own docs page (in the sidebar below) covers its content-prop shape, usage
examples, composition notes, and accessibility rationale. See `CONTRIBUTING.md` at the project
root for the conventions every pattern follows — in particular, patterns compose finished
components from `src/components/*` (a Story Card is a `Card` + `Image` + `Heading` + `Tag` +
byline text, not hand-rolled markup) rather than reimplementing them.

Built on the [Components](?path=/docs/components-overview--docs) layer above. The next layer,
[Templates](?path=/docs/templates-overview--docs), arranges these patterns into full page
layouts.
